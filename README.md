#### Viewing Time Lab
*Focus:* **File I/O, data cleaning, descriptive statistics, data visualizations, creating data reports.**

**Goal:** Explore results of a hypothetica A/B test conducted for a fictional company, called NotGPT, to examine whether changes to a slide format lead to changes in viewing time, and identify the likelihood of correlation.

**Approach:** Example two separate files, build a cleaning function to remove null values, run statistical analyses of data, produce data visualizations.

**Tools:** Python, statistics, Matplotlib.

**Impact:** Sucessfully cleaed null values, identified significant positive correlation between slide alteration and viewing time, visualized data.
